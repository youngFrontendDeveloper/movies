# Лучшие комедии для всей семьи

Веб-приложение — каталог семейных комедий с возможностью поиска, добавления и управления фильмами.

## Возможности

- **Поиск фильмов** — по названию, описанию, жанру или актерам
- **Каталог фильмов** — просмотр всех фильмов в виде карточек
- **Управление базой фильмов** — добавление, редактирование и удаление записей
- **Детальная страница фильма** — полная информация о каждом фильме
- **Рейтинговая система** — оценка фильмов по 5-звездочной шкале

## Технологии

- **Frontend:** React 18, React Router DOM 6
- **Стилизация:** Sass (SCSS)
- **Управление состоянием:** Redux, React Redux
- **Формы:** React Hook Form
- **База данных:** Firebase (Firestore)
- **Взаимодействие с Firebase:** React Firebase Hooks
- **Уведомления:** React Toastify

## Интерфейс

### Страница каталога фильмов
- Сетка карточек фильмов с постером, названием и краткой информацией
- Поисковая строка с фильтрацией в реальном времени
- Кнопка "Добавить новый фильм"
- На каждой карточке — кнопки редактирования и удаления

### Форма добавления/редактирования
- Поля: название, описание, жанр, актеры, ссылка на постер
- Звездный рейтинг
- Валидация всех полей
- Предпросмотр загружаемого постера

## Установка и запуск
### Настройка Firebase:
Создайте проект в Firebase Console
Создайте базу данных Firestore
Скопируйте конфигурационные данные
### Создайте файл .env в корне проекта:
.env
* REACT_APP_FIREBASE_API_KEY=your_api_key
* REACT_APP_FIREBASE_AUTH_DOMAIN=your_auth_domain
* REACT_APP_FIREBASE_PROJECT_ID=your_project_id
* REACT_APP_FIREBASE_STORAGE_BUCKET=your_storage_bucket
* EACT_APP_FIREBASE_MESSAGING_SENDER_I


### Запуск в режиме разработки:

bash
* npm install
* npm start
* Приложение будет доступно по адресу: http://localhost:3000
