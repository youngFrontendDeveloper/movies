1. Репозиторий для клонирования  https://github.com/youngFrontendDeveloper/movies
2. Для запуска проекта на локальном компьютере необходимо создать папку и с помощью терминала (cmd,PowerShell, Git Bush или др.) склонировать в нее данные из репозитория командой:  
git clone https://github.com/youngFrontendDeveloper/movies.git
3. После этого перейти в папку movies, которая появится после успешного клонирования проекта, командой: cd movies
4. Убедитесь, что Вы находитесь в ветке <Ваш путь до папки>/movies (main)
5. Установите необходимые зависимости командой: npm i
6. Убедитесь, что у Вас свободны порты  http://localhost:3000 и http://localhost:3001
7. После установки зависимостей запустите сервер командой: npm run server
8. При успешном запуске в окне терминала должно появиться приветствие сервера \{^_^}/ hi!  и информация о ресурсах   http://localhost:3000/movies
9. В другом окне терминала запустите проект командой: npm start  
10. В браузере по адресу http://localhost:3001 откроется окно с созданным проектом
